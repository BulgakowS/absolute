== Description ==

Using this plugin you can create multiple carousel in pages 
or posts both vertical and horizontal , you can give links to 
each carousel images  

following is list of features*

<p class=\"MsoNormal\"><span style=\"font-size: 14.0pt; mso-bidi-font-size: 11.0pt; line-height: 115%;\">***1. You can create multiple carousel with unique name and id, carousel name allow only capital and small letter.***</span><p class=\"MsoNormal\"><span style=\"font-size: 14.0pt; mso-bidi-font-size: 11.0pt; line-height: 115%;\">

2. You can also delete the carousel.***</span></p> <p class=\"MsoNormal\"><span style=\"font-size: 14.0pt; mso-bidi-font-size: 11.0pt; line-height: 115%;\">
3. You can upload different images to any carousel and can set image side order, also has the option to delete particular images from any carousel.</span></p> <p class=\"MsoNormal\"><span style=\"font-size: 14.0pt; mso-bidi-font-size: 11.0pt; line-height: 115%;\">
4. All images have a unique id.***</span></p> <p class=\"MsoNormal\"><span style=\"font-size: 14.0pt; mso-bidi-font-size: 11.0pt; line-height: 115%;\">
5. You can link the images to any URL and have option to update later. </span></p> <p class=\"MsoNormal\"><span style=\"font-size: 14.0pt; mso-bidi-font-size: 11.0pt; line-height: 115%;\">
6. You can change the image slide Order. </span></p> <p class=\"MsoNormal\"><span style=\"font-size: 14.0pt; mso-bidi-font-size: 11.0pt; line-height: 115%;\">
7. You can choose different setting for different carousel .</span></p> <p class=\"MsoNormal\" style=\"text-indent: .5in;\"><span style=\"font-size: 14.0pt; mso-bidi-font-size: 11.0pt; line-height: 115%;\">
a. Time delay in second
b. Number of images to scroll 
c. Animation speed
d. scroll right to left
e. wrap (end slide on first image, last image and both)
f. vertical slide
g. number of image to visible on frond end
h. Effect
i.  skin
j. Container width, height
k. image width and height

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
